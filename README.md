## Setup

```sh
git clone https://github.com/GPT-Engineer-App/nextjs-ssr-bot-auth.git
cd nextjs-ssr-bot-auth
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
